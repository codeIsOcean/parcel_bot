# Parcel Bot — Гид по экранам

## Как просмотреть все экраны

### WebApp (Parcel Bot v2.dc.html)
Откройте файл в браузере и кликайте по интерфейсу:

**Экраны отправителя:**
1. Главная → tab "📦 Отправить"
2. Выбор города → кнопка "Откуда" или "Куда"
3. Список попутчиков → кнопка "Найти попутчиков" или клик на маршрут
4. Форма отправки → клик на попутчика из списка
5. Чат → через нижнюю навигацию "💬 Чат"
6. Отслеживание → через навигацию "📦 Посылки" или клик на активную доставку
7. Профиль → через навигацию "👤 Профиль"
8. Оценка → на экране трекинга кнопка "⭐ Оценить"
9. Настройки → в профиле кнопка "⚙️ Настройки"
10. Подписка → в профиле перевозчика
11. Мои посылки → tab "Активные" / "Завершённые"

**Экраны перевозчика:**
1. Главная → tab "✈️ Перевезти"
2. Публикация рейса → карточка "Опубликовать рейс"
3. Входящие заявки → в списке заявок кнопки "Принять" / "Отклонить"

### Bot (Parcel Bot Telegram.dc.html)
Откройте файл и кликайте по inline кнопкам:

**Навигация:**
1. /start → выбор роли
2. "📦 Отправить посылку" → FSM создания заявки (7 шагов)
3. "✈️ Перевезти посылку" → меню перевозчика
4. "✈️ Опубликовать рейс" → FSM публикации (6 шагов)
5. "🔍 Найти попутчика" → список с inline кнопками
6. Клик на имя → профиль попутчика
7. "📬 Входящие заявки" → принять/торг/отклонить
8. "💬 Мои чаты" → список чатов → relay чат
9. "⭐ Оценить" → рейтинг 1-5 + комментарий
10. "👤 Профиль" → статистика + отзывы
11. "⭐ Подписка" → планы + Stars/TON оплата
12. "⚙️ Настройки" → язык + уведомления
13. Кнопка "/" → меню команд
