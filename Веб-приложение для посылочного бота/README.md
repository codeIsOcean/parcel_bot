# Parcel Bot — Посылочный сервис через попутчиков

## Описание
Сервис доставки посылок через пассажиров-попутчиков на международных рейсах.
Две версии: Telegram Mini App (веб-приложение) и Telegram Bot.

## Файлы проекта

### 1. `Parcel Bot v2.dc.html` — Telegram Mini App (WebApp)
Полноценное мобильное приложение в iPhone frame. Интерактивный прототип.

**Экраны:**
- Главная (Отправитель) — выбор маршрута, популярные маршруты, активные доставки
- Главная (Перевозчик) — публикация рейса, мои рейсы, входящие заявки
- Список попутчиков — фильтры (все/скоро/много кг/топ рейтинг), карточки с рейтингами
- Форма отправки посылки — описание, вес, размер, фото, предложение цены
- Публикация рейса — маршрут, дата, свободные кг, цена за кг, примечания
- Чат — relay-чат между отправителем и перевозчиком
- Отслеживание — таймлайн статусов (создана → принята → передана → в пути → доставлена)
- Профиль — рейтинг, доставки, отзывы, верификация
- Оценка доставки — 5 звёзд + комментарий
- Подписка — планы (1/3/12 мес), оплата через Telegram Stars и TON
- Мои посылки — вкладки активные/завершённые
- Настройки — язык (RU/EN/KZ), предложить маршрут, уведомления
- City Picker — модалка выбора города + ввод своего города

**Фичи:**
- Двуязычный интерфейс (RU/EN)
- Переключение ролей Отправитель/Перевозчик
- Рейтинги и отзывы
- Торг по цене (counter-offer)
- Верификация пользователей
- Telegram Stars + TON оплата подписки
- Bottom navigation (Главная, Посылки, Чат, Профиль)

### 2. `Parcel Bot Telegram.dc.html` — Telegram Bot версия
Интерактивный прототип бота в стиле реального Telegram чата.

**Экраны/Команды:**
- `/start` — приветствие + выбор роли
- Меню отправителя — Reply keyboard (Отправить, Найти, Посылки, Чаты, Профиль, Настройки)
- Меню перевозчика — Reply keyboard (Рейс, Заявки, Рейсы, Чаты, Профиль, Подписка)
- Создание заявки (FSM) — город → город → описание → вес → фото → цена → подтверждение
- Публикация рейса (FSM) — маршрут → дата → кг → цена → подтверждение
- Список попутчиков — inline кнопки, клик на имя → профиль
- Входящие заявки — принять/торг/отклонить
- Relay чат — конфиденциальный, контакты скрыты
- Рейтинг — 1-5 ⭐ + комментарий
- Профиль — свой и чужой с отзывами
- Подписка — планы + Telegram Stars + TON
- Настройки — язык, уведомления
- Мои посылки — список с статусами
- Мои рейсы — список с количеством заявок
- Список чатов

**Фичи бота:**
- FSM (Finite State Machine) для пошаговых сценариев
- Inline кнопки на сообщениях бота
- Reply keyboard для главных меню
- Кнопка `/` — меню команд (/start, /menu, /send, /profile, /settings)
- Toggle клавиатуры (показать/скрыть)
- Ввод текста через input + Enter
- Кнопка «Назад» на каждом экране (нет dead-ends)
- «Другой город» / «Другое» для свободного ввода
- Relay чат с скрытыми контактами
- Двуязычный (RU/EN)

## Технический стек для реализации

### Bot (Python + aiogram 3)
```
- aiogram 3.x — Telegram Bot Framework
- FSM (MemoryStorage / RedisStorage) — для пошаговых сценариев
- SQLAlchemy + PostgreSQL — база данных
- Redis — кеш, FSM storage
- aiogram-dialog (опционально) — для сложных меню
```

### Mini App (React + Telegram WebApp API)
```
- React 18+ / Next.js
- Telegram WebApp SDK (@twa-dev/sdk)
- TailwindCSS — стили
- Axios — API клиент
- React Router — навигация
```

### Backend API
```
- FastAPI / Django REST
- PostgreSQL — основная БД
- Redis — кеш, очереди
- TON SDK — приём TON платежей
- Telegram Bot Payments API — Stars
```

### Ключевые модели БД
```
Users — пользователи (роль, рейтинг, верификация)
Flights — опубликованные рейсы (маршрут, дата, кг, цена)
Parcels — заявки на посылки (маршрут, описание, вес, фото, цена)
Matches — совпадения заявка↔рейс
Messages — relay-чат сообщения
Reviews — отзывы и рейтинги
Subscriptions — подписки перевозчиков
Payments — платежи (Stars/TON)
Cities — города и маршруты
RouteVotes — голосование за новые маршруты
```

### Relay Chat система
- Сообщения проходят через бота
- Контакты скрыты до подтверждения обеими сторонами
- Screen Manager контролирует состояние чата

### Рейтинговая система
- После каждой доставки — взаимная оценка (1-5 ⭐ + текст)
- Средний рейтинг отображается в профиле
- Верификация паспорта для повышения доверия
